# F2E--week1 台灣走走

台灣旅遊景點網站

### 簡介
模擬常見的旅遊網站，可透過搜尋按鈕、類別圖示，去找到你所想要的景點。
### 網站功能

v1.0
1. 依照不同活動/類別連結找到對應的結果
2. 景點詳細內容包含google地圖，方便查看對應的景點位置


v1.5
1. 將資料整理至vuex
2. 新增網站loading效果

v2.0 (進行中)
1.  添加搜尋鄰近地區的功能


### 使用技術

1. Vue3
2. Vue CLI
3. Vue Router
4. Axios
5. Scss
6. Bootstrap
7. vuex
